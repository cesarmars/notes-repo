# Processor 
An electronic components that executes operations on an external data source like memory. Examples of processors found in 
computers

- central processing unit, cpu
- graphics processing unit, gpu
- accelerated processing until, apu

# CPU
The brain of the computer executes instructions, manages software data, manages memory, and performs logical operations and
more. 

Cores are processors that live inside the cpu. It each processor/core are independently working CPUs that perform 
general purpose instructions at given time. 

CPU's are designed around an instructional set architecture (ISA). ISA are hardwired to a specific blueprint that define 
their registers, instructional operations, how memory is accessed and how instructions encode binaries.

# Instructional Set Architecture (ISA)

Intructional Set Architecture is how software interfaces with hardware through defined registers, memory model, data types,
and instructions. There is two major ISA's, Complex Instruction Set Computer (CISC), and Reduced Instructional 
Set Computer (RISC-V)

Different CPU's implement different ISA's

x86(CISC) = Intel and AMD processors found in consumer-grade computers.
ARM = Apple silicon series, apple computers.
RISC-V = embedded systems, consoles, cellphone etc.

All of these ISA's have their assembly language that they follow, according to their ISA.

For example, RISC-V uses RISC-V assembly language, that is: lw a0 0(sp) etc..

x86 uses x86 assembly language.
